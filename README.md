# ElectroServe

ElectroServe is an appliance service booking and technician management platform.
This first MVP focuses on the customer booking flow: selecting a company,
choosing an appliance, entering service details, matching technicians, and
confirming a booking.

## Tech Stack

- React
- Vite
- CSS

## Run Locally

```bash
npm install
npm run dev
```

## Available Scripts

- `npm run dev` starts the local Vite server.
- `npm run build` creates a production build.
- `npm run lint` checks the project with ESLint.
